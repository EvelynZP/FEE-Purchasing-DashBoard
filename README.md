# FEE-Purchasing-DashBoard

### 项目描述
此看板使用部门是采购中心，为采购人员便于查看近期新发地的市场价格，5个常购品类的实采情况,以便实时<br>
__数据来源__：<br>
     1.采购部门提供的历史采购情况<br>
     2.利用Python实时爬取和处理的新发地菜品价格
